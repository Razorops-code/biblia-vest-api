# Biblia Vest - API

### Project layers structure
![Architecture](/architecture.png)

**How to run**
```sh

git clone https://github.com/juniorstreichan/biblia-vest-api.git
cd biblia-vest-api
yarn install

# run tests
yarn test

# generate swagger docs
yarn swagger

# for development
yarn dev

# build the app
yarn build

# start production 
yarn start

```
